# da-hack-me-ai


# Setup

## UI

```
curl -fsSL https://get.pnpm.io/install.sh | sh -
```

## API
```
curl -LsSf https://astral.sh/uv/install.sh | sh
```


# Run

## UI

```
pnpm run dev
```

## API

```
uv run main.py
```
